# Olist Brazilian E-Commerce Analysis

## Project Description
This project performs exploratory data analysis on the Olist Brazilian E-Commerce dataset using Python. The objective is to identify sales trends, top-performing categories, regional performance, and customer review patterns.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Data Transformation
4. Dataset Merging
5. Exploratory Data Analysis
6. Visualization
7. Dashboard KPI Generation

## Visualizations
- Revenue by Category
- Monthly Sales Trend
- Revenue by State
- Order Value Distribution
- Review Score Distribution
- Revenue Heatmap

## Key Outcomes
- Identified top-performing product categories
- Analyzed monthly revenue patterns
- Examined regional sales performance
- Evaluated customer satisfaction through review scores
